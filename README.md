with the following structure -
```

column                 type
------------------------------
name                    varchar(20)
mobile                  varchar(20)
email                   varchar(20)
address                 varchar(20)
count                   int

```
 
```
     docker run --name benchmark \
     -e MYSQL_HOST=<MYSQL HOST> \
     -e MYSQL_PORT=<MYSQL PORT> \
     -e MYSQL_USER=<MYSQL_USER> \
     -e MYSQL_PASS=<MYSQL PASS> \
     -e MYSQL_DB=<MYSQL DB> \
     -d vivekcs0114/php-benchmark ```
